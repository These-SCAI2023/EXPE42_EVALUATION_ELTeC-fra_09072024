Les Expériences et résultats contenus dans ce dépôt sont relatifs aux travaux de thèses de doctorat de Caroline Koudoro-Parfait relatifs à la question de l'impact du bruit de l'OCR sur les résultats de la tâche de REN.

Dans le dossier prog vous retrouvez les scripts pour l'évaluation avec les métriques : cosinus, jaccard, le CER et WER.

Le corpus est disponible ici : https://github.com/These-SCAI2023/EXPE44_CORPUS_EVAL/tree/main
